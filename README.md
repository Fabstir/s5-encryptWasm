# S5 encryptWasm for S5 Client Javascript SDK

An S5 encryptWasm for S5 Client in Javascript for Decentralized content-addressed storage network S5.

## Installing

1. Clone this repo.
2. cd s5-encryptWasm
3. yarn install
4. yarn wasm-pack
5. yarn build
